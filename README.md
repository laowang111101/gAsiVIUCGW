# 网上花店商城系统 python857

本项目是一款基于前后端分离模式的网上花店商城系统，采用以下技术栈进行开发：

## 技术栈

- **后端：**
  - Python3.9
  - Django
  - Django REST framework (DRF)
- **数据库：**
  - SQLite
- **前端：**
  - Vue3
  - Element Plus
- **Node.js版本：**
  - v20.10
- **开发工具：**
  - PyCharm
  - WebStorm

## 功能模块

- 用户认证
  - 登陆注册
- 商品管理
  - 鲜花管理
- 订单管理
  - 鲜花订购
  - 订单管理

- 系统管理员
- 注册用户

- 后端：Python3.9及以上版本环境
